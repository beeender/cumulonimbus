Cumulonimbus allows you to display your site's tags on the surface of a rotating sphere.


== Description ==

This is a similar plugin with WP-Cumulus (http://wordpress.org/extend/plugins/wp-cumulus/) but implemented with HTML5.

This plugin requires the modern browsers which support HTML5.

== Installation ==

1. Download the zip file and extract the contents.
2. Upload the 'cumulonimbus' folder to your plugins directory (wp-content/plugins/).
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Find a place for this widget in appearance->widgets.
